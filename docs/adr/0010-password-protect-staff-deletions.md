# Destructive staff actions require one shared password

Hosting MEI MEI publicly makes its Server Functions reachable beyond the booth, so the offline kiosk's hidden gesture is no longer sufficient protection. Deleting one Response or all Responses requires a shared staff password stored as Vercel's `ADMIN_PASSWORD`, while taking the quiz remains account-free; unlocking Staff Mode lasts only in the current tab's memory and a refresh or tab close relocks it. Full user accounts, roles, and persisted admin sessions would add complexity without improving a one-fair moderation workflow.
